# HabitSwap (React Native / Expo)

Eine komplette React-Native-App zur funktionalen Habit-Analyse und zum gezielten Austausch von schlechten gegen gute Gewohnheiten.

## Features

- Separates Erfassen von **schlechten** und **guten** Habits.
- Metriken je Habit:
  - Cue / Trigger
  - Reward
  - Impact / Nutzen (0-10)
  - Zeit (Minuten/Tag)
  - Schwierigkeit (0-10)
  - Motivation (0-10)
  - Opportunity (0-10)
- Priorisierung via Score auf Basis von Fogg + COM-B inspirierten Dimensionen.
- Automatische Vorschläge für Habit-Swaps (ähnlicher Cue/Reward, realistische Zeitnähe).
- Dynamisches Hinzufügen und Löschen von Habits.

## Start

```bash
npm install
npm run start
```

Für Web:

```bash
npm run web
```
